# Contributing to AI-Bot-Agent

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/kavyahj04/AI-Bot-Agent`
3. Install dependencies: `pip install -r requirements.txt`
4. Create a `.env` file with the required environment variables (see README)
5. Start the server: `uvicorn app:app --host 0.0.0.0 --port 5000`

## Making Changes

- Create a new branch: `git checkout -b feature/your-feature-name`
- Make your changes and commit: `git commit -m "description of change"`
- Push and open a Pull Request

## Code Style

- Follow PEP8 Python style guidelines
- Keep route handlers small and focused
- Document any new environment variables in the README

## Reporting Issues

Open a GitHub Issue with a clear title and description of the bug or feature request.
